# AppSignals SDK Release Notes

## v1.0.0
Release date: 2026-01-05  
Git tag: v1.0.0

### Highlights
- Event tracking with batching and retry.
- Crash reporting via PLCrashReporter.
- Session replay wireframes with redaction.
- Privacy manifest included.

### Compatibility
- iOS 16+
- Swift 6 toolchain

### Notes
- The SDK supports both local Swift Package references and Git-hosted dependencies.
